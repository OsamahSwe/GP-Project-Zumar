# Zumar (زمر) - University Clubs Management System

A modern React.js application for university club management with Firebase integration.

## Features

- Beautiful homepage with purple gradient design
- Authentication system (Login/Signup)
- Real-time form validation with Arabic support
- Firebase integration for backend services
- Responsive design with KSU blue/white theme

## Technology Stack

- React.js 18
- Firebase Authentication & Firestore
- CSS3 with modern features
- React Router v6

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Configure Firebase in `src/config/firebase.js`
4. Start the development server: `npm start`
